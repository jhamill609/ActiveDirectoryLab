<h1>Active Directory Lab</h1>


<h2>Description</h2>
Created a virtual Domain Controller to simulate a corporate environment. Deployed Active Directory Domain Services, DNS, and DHCP to provide centralized authentication, name resoultion, and IP address management. Used PowerShell to automate user account creation and password assignment. Enabled client machines to join and authenticate to the domain. 
<br />


<h2>Technologies and Tools used</h2>

<h4>Infrastructure Services</h4>
- <b>Active Directory Domain Service, DNS, DHCP</b>
<h4>Networking</h4>
- <b>Default Gateway configuration, IP addressing, domain-joined clients</b> 
<h4>Utilities</h4>
- <b>Powershell, ipconfig, ping</b>

<h2>Oracle Virtualbox</h2>

- <b>Windows 11</b>
- <b>Windows Server 2022</b>
